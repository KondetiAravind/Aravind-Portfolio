# 🌐 Kondeti Aravind — Personal Portfolio Website

A modern, recruiter-focused portfolio website built using **Next.js 16**, **TypeScript**, and **Tailwind CSS**, showcasing my projects, research interests, internship experience, technical skills, and academic journey at **Indian Institute of Technology Bhubaneswar**.

The portfolio serves as a centralized hub for recruiters, collaborators, and researchers to explore my work across **Software Engineering, Computer Vision, NLP, Data Engineering, and Artificial Intelligence**.

## ⚙️ Local Setup

Clone repository:

```bash
git clone https://github.com/KondetiAravind/Aravind-Portfolio.git

cd Aravind-Portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## 🌐 Deployment

### Vercel Deployment

```bash
git add .
```

Vercel automatically deploys the latest version from the main branch.
